## Build Rules (NO EXCEPTIONS)

1. NO em dashes in code or comments - use regular hyphens only
2. Reference DosePair everywhere (never other names)
3. Import from design-system/components/ for all UI elements
4. All colors from design-system/tokens/ CSS variables
5. All fonts: Space Grotesk (headings), IBM Plex Sans (body), IBM Plex Mono (numbers/doses/times)
6. Disclaimer component on every results screen (always last)
7. Row limits: 15 desktop / 5 mobile
8. Backend never exposes ANTHROPIC_API_KEY to client
9. Version every new file (v1, v2, v3). Never overwrite.
